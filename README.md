#

```bash OpenClaw Telegram Bot with 9Router

Production-ready Telegram AI bot using OpenClaw and 9Router.

## Features

- Telegram bot integration
- Multi-model routing with 9Router
- OpenAI-compatible API
- Docker-based deployment
- Persistent conversation memory
- Automatic restart and failover

## Architecture

Telegram → OpenClaw → 9Router → OpenRouter / Claude / Gemini / OpenAI

## Requirements

- Ubuntu 22.04+
- Root access
- Telegram bot token
- VPS (Tencent Cloud recommended)
